# Supervised Learning
 **Definition:**

A type of machine learning where the model is trained using labeled data.

"Labeled" means each input (X) has a correct output/answer (Y).

**Statement:**

The model learns the relationship between X (features) and Y (target), so it can predict Y for new X.
